# Healthcare-Outcomes-Forecasting-KPMG-Collaboration-UCF-2025-
Partnered with KPMG and PhD researchers to forecast COPD and CHD healthcare needs using Florida AHCA (Agency for Health Care Administration) patient-level data and time series models. Built reproducible Alteryx pipelines and Tableau dashboards to support healthcare resource planning and quality improvement decisions. Contributed to publication-ready findings and data documentation ensuring transparency and reproducibility.
